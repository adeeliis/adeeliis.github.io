En este apartado encontrarás información sobre HTML:


HTML o Lenguaje de Marcado de Hipertexto es un lenguaje de programación utilizado para crear y estructurar páginas web. A través de HTML, podemos definir la estructura básica de una página, incluyendo el texto, imágenes, enlaces, tablas y otros elementos que componen una web. HTML utiliza "etiquetas" para indicar cómo debe mostrarse y organizarse el contenido en el navegador. Es el primer paso en la creación de cualquier página web y funciona en conjunto con otros lenguajes como CSS (para el diseño).


Principales usos:

  1. Estructurar el contenido web: HTML permite definir la estructura de una página web, como encabezados, párrafos, listas, enlaces, imágenes y otros elementos.
  2. Crear páginas web estáticas: HTML es fundamental para construir sitios web simples y estáticos, aunque con el tiempo suele combinarse con CSS y JavaScript para hacerlos más interactivos.
  3. Bases para otros lenguajes: HTML es la base sobre la cual se construyen otros lenguajes web como CSS para estilo visual y JavaScript para funcionalidades interactivas.
  4. Crear formularios: HTML permite incluir formularios donde los usuarios pueden ingresar información, como encuestas o registros.

Algunas etiquetas basicas de HTML

  1. <html> Define el inicio de un documento HTML.
  2. <head> Contiene metadatos sobre el documento, como el título, los enlaces a archivos CSS y otros elementos no visibles para el usuario.
  3. <title> Establece el título de la página, que aparece en la pestaña del navegador.
  4. <body> Contiene el contenido visible de la página web, como texto, imágenes y enlaces.
  5. <h1> Son etiquetas de encabezado que definen títulos de diferentes niveles. <h1> es el encabezado principal, y los números indican subniveles.
  6. <p> Define un párrafo de texto.
  7. <a> Crea un enlace a otra página o recurso.
  8. <img> Muestra una imagen en la página. Se utiliza el atributo src para definir la fuente de la imagen.
  9. <ul> y <ol> Son etiquetas para crear listas. <ul> crea una lista no ordenada y <ol> una lista ordenada.
  10. <li> Define un ítem dentro de una lista.
